# Project Summary
The "Smart Workout Tracker" is a web application designed to help users manage their workout routines effectively. Built with pure HTML, CSS, and JavaScript, it features a user-friendly interface that allows users to log their exercises, track their progress, and receive adaptive weight recommendations based on their workout history. The application emphasizes a modern aesthetic with a glassmorphism design, enhancing user engagement through visual appeal.

# Project Module Description
The application consists of three main modules:
1. **Structure**: The HTML layout includes a header, day tabs for different workout routines, and a grid for exercise cards.
2. **Styling**: Utilizes CSS for a glassmorphism effect, responsive design, and light/dark themes.
3. **Logic**: JavaScript handles data management, adaptive weight suggestions, plateau detection, and localStorage interactions.

# File Description Inventory
- **index.html**: Contains the main structure of the application, including headers, navigation, and modals.
- **style.css**: Manages the visual design, including themes and layout.
- **script.js**: Implements core functionality such as logging workouts, adaptive weight suggestions, and localStorage management.
- **package.json**: Lists project dependencies and scripts for building and running the application.
- **template_config.json**: Configuration settings for the application templates.

# Technology Stack
- **Frontend**: HTML5, CSS3, Vanilla JavaScript (ES6+)
- **Fonts**: Google Fonts (Inter)
- **Images**: Unsplash for exercise visuals

# Usage
To set up and run the application:
1. Install dependencies:
   ```bash
   pnpm install
   ```
2. Build the project:
   ```bash
   pnpm run build
   ```
3. Start the application:
   ```bash
   pnpm run start
   ```
